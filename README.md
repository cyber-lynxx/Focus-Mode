# Focus Mode
If you have trouble resisting the urge to click that YouTube shortcut, this website removes the visual temptations, while also showing the time and allows you to travel to other websites using an input form.

### Inspiration
I got the idea for this project from my experiences with trying to do work with a computer open. While there is no reason for it to be open while you are trying to read a book, it can be useful for purposes like telling time (although maybe just get a watch?) and researching things you don't know. 

### Explanation
So I made a webpage with a green background, which makes it more comfortable to look at, and has links that you MIGHT need, and in case that's not enough (which it probably won't be), there is an input box where you input a link and it opens that in the new tab to avoid you seeing that tempting logo. And don't worry, you won't be able to access social media from that, you'll get an alert telling you to touch grass. 

If you are the type of person who enjoys having a fidget, there is a built in clicker fidget, but to avoid you getting distracted, there is a threshold at 500 clicks, where an alert tells you to refocus. The threshold is that high because if it were lower it would accidentally lecture someone innocently clicking it.

There is also a 30 minute timer, and when it is finished, it suggests that you take a break.
